Agrupación de elementos

Para agrupar elementos como si fueran un solo bloque utilizamos la etiqueta <div> </div>.
Suele utilizarse para segmentar mejor la información, aplicar estilos CCS o manipular varios elementos con JavaScript.
Ejemplo:

Energias Renovables

Las energias renovables son importantes para crear un ambiente sostenible que pueda brindar beneficios tanto al ahora como al mañana.

Creación de Listas Desordenadas

Para crear listas no ordenas debemos usar las etiquetas <ul> </ul> - viñetas - y dentro de estas solo pueden haber elementos <li> </li>.

A continuación un ejemplo de su uso

  • Huevos
  • Pan
  • Cebolla
  • Tomate

Creación de Listas Ordenas

Para crear listas ordenas debemos usar las etiquetas <ol> </ol> - numerados - y dentro de estas solo pueden haber elementos <li> </li>.

Podemos usar atributos como "start" y "type" para definir el comienzo y estilo de la numeración correspondiente.

A continuación un ejemplo de su uso con estilo de letras y que comienza en el tercer caracter.

  1. Huevos
  2. Pan
  3. Cebolla
  4. Tomate
