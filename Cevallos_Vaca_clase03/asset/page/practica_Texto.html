Definicion de un parrafo de texto

Ussamos la etiqueta <p> para escribir un parrafo. En caso de querer escribir varios parrafos debemos abrir y cerrar la etiqueta para cada parrafo que queramos. Ejm:

<p> Contenido del parrafo numero 1 </p>

<p> Contenido del parrafo numero 2 </p>

Insertar un salto de linea

Para poder insertar saltos de linea dentro de un parrafo usamos la etiqueta <br>. Tenga en cuenta que no necesita cerrarla. Ejm:

SINTAXIS: <p> Primera linea <br> Segunda linea </p>. A continuación el como quedaría el código.

Primera linea
Segunda lina
Tercera linea
Etc...

Tipos de encabezados

h1: Encabezado de primer nivel

h2: Encabezado de segundo nivel

h3: Encabezado de tercer nivel

h4: Encabezado de cuarto nivel

h5: Encabezado de quinto nivel
h6: Encabezado de sexto nivel

Enfasis en una palabra o frase

Para darle enfasis a una palabra o frase usamos las etiquetas <em> </em>. En otras palabras, para poner cursiva. Ejm:

Hola, mi nombre es Julio Cevallos es un gusto estar con ustedes el día de hoy.

Indicar importancia o relevancia

Para darle relevancia a una palabra o frase con respecto al resto del texto usamos las etiquetas <strong> </strong>. En otras palabras, para poner negrita. Ejm:

Hola, mi nombre es Julio Cevallos es un gusto estar con ustedes el día de hoy.

Etiquetas para insertar estilos CSS

Para darle estilos CSS o manipular con JavaScript a una palabra o frase con respecto al resto del texto usamos las etiquetas <span> </span>. Ejm:

Hola, mi nombre es Julio Cevallos es un gusto estar con ustedes el día de hoy.

Información extra

Tenga en cuenta que existen etiquetas como $lti&ht </i> y <b> </b> para poner letras cursivas y negritas respectivamente, no obstante, solo tienen un uso visual más no semánticos como dar importancia o enfasis
