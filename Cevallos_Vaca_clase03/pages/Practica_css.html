Practica de CSS

CSS nos ayuda/sirve para dar estilo y diseño a nuestra Página Web.
Para apricarlo tenemos 3 opciones:

1.- En línea - código embebido
style="color:red">Línea de Estilo con código embebido

2.- Código Interno - Código en la cabecera
<style>
p{
background:blue;
}
</style>

3.- Externo - File .CSS
<link rel="stylesheet" href="estilo.css">

Modelo Cajas#1:
-Todo elemento de HTML se debe tratar como una caja.

Modelo Cajas#2:
-Todo elemento de HTML se debe tratar como una caja.

Notas:

1. Siempre se ejecuta de arriba hacia abajo, es decir, la misma etiqueta la puede escribir dos veces y esta presentara en pantalla ek último efecto programado.

2. Tener cuidado con montar las etiquetas en el diseño.

3. Valide7Resetear antes de aplicar su diseño. Dependiendo de su diseño.

Hipervinculos

Hipervinculos: abrir en una nueva pestaña

Abrir un hipervínculo en una nueva pestaña es una función que permite que al hacer clic en un enlace, la página web de destino se abra en una nueva pestaña del navegador en lugar de reemplazar la página actual.
Abrir en una pestaña nueva

Hipervinculos: abrir en la misma pestaña

Un hipervínculo es un enlace a otra ubicación, que puede ser una página web, un documento o una sección del mismo documento. "Abrir en la misma pestaña" es una opción que indica que al hacer clic en el enlace, el contenido de destino se cargará en la pestaña actual del navegador, reemplazando el contenido original en lugar de abrirse en una pestaña nueva.
Abrir en la misma nueva

Hipervinculos: abrir en una nueva pestaña, modo seguro

Un hipervínculo es un enlace que dirige a otra página, documento o recurso. "Abrir en otra pestaña" es una instrucción para que ese enlace se abra en una nueva pestaña del navegador en lugar de reemplazar la página actual. "Modo seguro" se refiere a una configuración del navegador que carga la página sin complementos ni extensiones, lo que puede ser útil para solucionar problemas o prevenir riesgos de seguridad.
Abrir en la misma nueva (Modo Segura)
