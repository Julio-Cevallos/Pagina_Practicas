<code> — código en línea

Definición: la etiqueta <code> se usa para marcar fragmentos de código o comandos.

Uso: aquí mostramos una instrucción JavaScript dentro de <code>: console.log('Hola mundo');

<var> — variables

Definición: la etiqueta <var> representa el nombre de una variable o parámetro en texto técnico.

Uso: la variable x contiene el valor 10, por lo tanto console.log(x); mostrará 10.

<mark> — resaltar texto

Definición: la etiqueta <mark> se usa para resaltar o enfatizar texto, como si fuera una marca amarilla.

Uso: Presta atención a este texto resaltado que indica un punto importante.

<small> — texto secundario

Definición: la etiqueta <small> reduce visualmente el tamaño del texto y se usa para notas, disclaimers o metadatos.

Uso: Nota: Este es un ejemplo de nota al pie mostrada con <small> — ideal para aclaraciones cortas.
